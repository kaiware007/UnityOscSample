# UnityOscSample
OSC Sample for Unity & VVVV

## Require
- VVVV
- Unity2018.2.21f1 or later

## Usage
- Open vvvv/OscTest.v4p
- Open Assts/Scenes/SamplesScene.unity
- Play Scene
- Push **SendBang** in OscTest.v4p
- Display OSC Message in Console
